# Collect Bootstrap Trees

Collect Bootstrap Trees

## Usage

``` r
collect_bootstraps(
  bs_dir,
  output_dir = bs_dir,
  prefix = "cactus_ALL_bootstraps"
)
```

## Arguments

- bs_dir:

  Character. Directory where bootstraps chunks are stored.

- output_dir:

  Character. Directory to write concatenated bootstraps. Defaults to
  bs_dir.

- prefix:

  Character. Prefix to use. Defaults to "cactus_ALL_bootstraps".

## Value

Path to concatenated trees file.
