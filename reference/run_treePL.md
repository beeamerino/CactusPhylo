# Run treePL via Wrapper

Cites the reference treePL wrapper:
https://github.com/tongjial/treepl_wrapper

## Usage

``` r
run_treePL(cfg, treefile, label, wrapper_sh)
```

## Arguments

- cfg:

  Character. Configuration file.

- treefile:

  Character. Tree file.

- label:

  Character. Label for the run.

- wrapper_sh:

  Character. Path to the treepl wrapper shell script.
