# Comparative Tree space validation pipeline

Comparative Tree space validation pipeline

## Usage

``` r
validate_phylogenies(
  trees_mapping_list,
  checklist_csv,
  constraints_map,
  out_dir
)
```

## Arguments

- trees_mapping_list:

  Named list of trees.

- checklist_csv:

  Character. Spellings check list CSV.

- constraints_map:

  Character. Taxonomy mapping table.

- out_dir:

  Character. Destination folder.

## Value

Generates tree distance results and tree-space coordinates.
